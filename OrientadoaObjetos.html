Diagrama de Clases

Detalles de las Clases

Clase Persona

Atributos: - id: int, - nombre: String, - apellido: String, - email: String

Métodos: + getId(): int, + getNombre(): String, + setNombre(nombre: String): void, + getEmail(): String, + setEmail(email: String): void

Funcionalidad: Clase base de la que heredan Usuario y Bibliotecario.

Clase Usuario (Extiende Persona)

Atributos: - idUsuario: int, - historialPrestamos: ArrayList<Prestamo>

Métodos: + consultarHistorial(): void, + devolverLibro(libro: Libro): void, + solicitarPrestamo(libro: Libro): void

Funcionalidad: Representa a los usuarios de la biblioteca y gestiona sus préstamos.

Clase Bibliotecario (Extiende Persona)

Atributos: - idBibliotecario: int

Métodos: + procesarPrestamo(usuario: Usuario, libro: Libro): void, + procesarDevolucion(usuario: Usuario, libro: Libro): void, + gestionarCatalogo(): void

Funcionalidad: Representa al personal que gestiona los préstamos, devoluciones y el catálogo.

Clase Notificacion

Atributos: - idNotificacion: int, - mensaje: String, - fecha: Date, - tipo: String (Recibido, Mulla, Información General)

Métodos: + enviarNotificacion(usuario: Usuario): void

Funcionalidad: Se encarga de enviar notificaciones a los usuarios.

Clase Prestamo

Atributos: - idPrestamo: int, - fechaPrestamo: Date, - fechaDevolucionPrevista: Date, - fechaDevolucionReal: Date, - activo: boolean, - usuario: Usuario, - libro: Libro

Métodos: + calcularMulta(): double, + extenderPrestamo(dias: int): void, + finalizar(): void

Funcionalidad: Gestiona el préstamo, cálculo de multa y devolución del libro.

Clase Libro

Atributos: - idLibro: int, - titulo: String, - autor: String, - editorial: String, - año: int, - disponible: boolean

Métodos: + getTitulo(): String, + estaDisponible(): boolean

Funcionalidad: Contiene la información del libro y su disponibilidad.

Clase Almacen (Catalogo)

Atributos: - libros: ArrayList<Libro>

Métodos: + agregarLibro(libro: Libro): void, + eliminarLibro(libro: Libro): void, + verificarDisponibilidad(idLibro: int): boolean, + buscarPorIdLibro(idLibro: int): Libro, + listarDisponibles(): ArrayList<Libro>

Funcionalidad: Gestiona el catálogo de libros, permitiendo agregar, eliminar y verificar disponibilidad.

FUNCIONALIDAD GENERAL DEL SISTEMA

Este sistema gestiona usuarios, bibliotecarios, préstamos y el catálogo de libros. Los usuarios pueden solicitar préstamos, devolver libros y recibir notificaciones, mientras que los bibliotecarios gestionan los préstamos, devoluciones y el catálogo.
