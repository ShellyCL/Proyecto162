Modelo Ambiental

Declaración de Propósitos

El sistema de gestión de bibliotecas tiene como propósito mejorar la eficiencia en la administración de recursos bibliográficos, facilitando el proceso de préstamo y devolución de libros, permitiendo una gestión de inventario en tiempo real y asegurando un acceso ágil a la información para usuarios y bibliotecarios.

  • Optimizar la administración de libros y usuarios.
  • Permitir el acceso rápido a la información sobre disponibilidad de libros.
  • Registrar usuarios y gestionar sus préstamos de manera efectiva.
  • Mejorar el Acceso a Recursos Académicos
  • Optimizar la Gestión de Préstamos y Devoluciones
  • Automatizar el Registro y Control de Inventario
  • Generar Reportes de Uso y Estadísticas
  • Mejorar la Experiencia del Usuario
  • Integrar con Otros Sistemas Universitarios

Diagrama de Contexto

Lista de Acontecimientos

  • Un lector se registra en el sistema proporcionando sus datos personales.
  • Un lector solicita el préstamo de un libro o material bibliográfico.
  • El lector devuelve el material bibliográfico prestado.
  • Un lector o bibliotecario verifica si un libro específico está disponible.
  • El lector solicita extender el período de préstamo de un libro.
  • El bibliotecario o director solicita informes sobre el estado de la biblioteca (préstamos activos, libros disponibles, etc.).
  • El bibliotecario agrega un nuevo libro o material al sistema.
  • El bibliotecario da de baja o elimina un libro del inventario.
  • El sistema envía una notificación al lector recordándole una devolución próxima o atrasada.
  • El sistema bloquea al lector por retrasos prolongados o acumulación de multas.
