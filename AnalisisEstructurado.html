Modelo Ambiental

Declaraciones de Propósitos

El sistema de gestión de bibliotecas tiene como propósito mejorar la eficiencia en la administración de recursos bibliográficos, facilitando el proceso de préstamo y devolución de libros, permitiendo una gestión de inventario en tiempo real y asegurando un acceso ágil a la información para usuarios y bibliotecarios.

  • Optimizar la administración de libros y usuarios.
  • Facilitar el préstamo y devolución de libros.
  • Permitir el acceso rápido a la información sobre disponibilidad de libros.
  • Registrar usuarios y gestionar sus préstamos de manera efectiva.

Diagrama de Contexto

El diagrama de contexto muestra las interacciones del sistema con las entidades externas. En este caso, el sistema de biblioteca interactúa con usuarios, bibliotecarios y otros sistemas relacionados (por ejemplo, sistemas de proveedores de libros).

  • Un nuevo usuario se registra en el sistema.

Modelo de Comportamiento

Diagrama de Flujo de Datos (DFD)

El Diagrama de Flujo de Datos (DFD) representa cómo los datos se mueven entre los diferentes procesos del sistema de biblioteca. Los principales flujos incluyen el registro de usuarios, la solicitud de préstamos y la devolución de libros.

  • Entrada: Un usuario solicita un libro.
  • Proceso: El bibliotecario procesa la solicitud y registra el préstamo en el sistema.
  • Salida: El sistema actualiza la disponibilidad del libro y asigna el préstamo al usuario.

Diagrama Entidad-Relación (ERD)

El Diagrama Entidad-Relación (ERD) muestra las entidades clave del sistema de biblioteca y sus relaciones. Las principales entidades son:

  • Usuario: Representa a la persona que utiliza el sistema, con atributos como nombre, dirección y número de usuario.
  • Libro: Contiene los detalles del libro, como título, autor, ISBN y estado de disponibilidad.
  • Préstamo: Una relación que conecta a un usuario con un libro prestado. Contiene atributos como la fecha de préstamo y la fecha de devolución esperada.

Diccionario de Datos

El diccionario de datos describe los principales campos y entidades que se utilizan en el sistema de biblioteca:

  • Usuario: Persona que utiliza el sistema. Campos: nombre, dirección, teléfono, correo electrónico, número de usuario.
  • Libro: Detalles de los libros disponibles en la biblioteca. Campos: título, autor, ISBN, categoría, estado (disponible, prestado).
  • Préstamo: Información sobre los libros prestados. Campos: usuario, libro, fecha de préstamo, fecha de devolución esperada, fecha de devolución real.
  • Bibliotecario: Usuario del sistema con permisos administrativos para gestionar préstamos y devoluciones.
