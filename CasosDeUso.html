Casos de uso

Caso de Uso: Préstamo
CASO: Préstamo
Actores: Bibliotecario, Usuario
Tipo: primario y esencial
Descripción:
  • - El usuario solicita un libro específico.
  • - El bibliotecario verifica la disponibilidad del libro en el sistema.
  • - Si el libro está disponible, el bibliotecario solicita los datos del usuario para registrar el préstamo.
  • - El sistema marca el libro como "no disponible" y guarda el registro del préstamo con las fechas de inicio y devolución prevista.
Caso de Uso: Devolución
CASO: Devolución
Actores: Bibliotecario, Usuario
Tipo: primario
Descripción:
  • - El usuario regresa un libro que tiene en préstamo.
  • - El bibliotecario busca el registro del préstamo en el sistema y verifica que el libro corresponde al usuario.
  • - El sistema calcula si existe alguna multa en caso de retraso.
  • - El bibliotecario marca el préstamo como finalizado y actualiza el estado del libro a "disponible".
Caso de Uso: Disponibilidad de Libro
CASO: Disponibilidad de Libro
Actores: Usuario
Tipo: Secundario
Descripción:
  • - El usuario consulta la disponibilidad de un libro en el catálogo de la biblioteca.
  • - El sistema muestra el estado del libro ("disponible" o "no disponible") junto con información adicional (autor, título, ubicación).
  • - Si el libro está disponible, el usuario puede proceder a solicitar el préstamo.
Caso de Uso: Administración de Usuarios
CASO: Administración de Usuarios
Actores: Bibiotecario
Tipo: Secundario
Descripción:
  • - El bibliotecario agrega nuevos usuarios o actualiza la información de usuarios existentes.
  • - El sistema almacena los datos personales y el historial de préstamos de cada usuario.
  • - El bibliotecario también puede consultar el historial de préstamos de un usuario para verificar su estado y sanciones, si aplica.
Caso de Uso: Administración de Libros
CASO: Administración de Libros
Actores: Bibiotecario
Tipo: Secundario
Descripción:
  • - El bibliotecario puede agregar, editar o eliminar libros del catálogo de la biblioteca.
  • - El sistema permite registrar la información del libro (título, autor, editorial, año, etc.) y actualizar su estado de disponibilidad.
  • - Esta funcionalidad es utilizada para mantener el catálogo organizado y actualizado.
